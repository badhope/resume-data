## ⚡ Quick Start

### Prerequisites

- Trae IDE v0.1.0 or higher
- A tolerance for relentless opposition

### Installation

```bash
# Simply have this in your .trae/skills directory
# It activates automatically when you debate or want contrarian responses
```

### Activation

The skill activates **automatically** when:
- You make any declarative statement
- You explicitly ask to debate or argue
- You invoke via: `/skills:argumentative-personality`
