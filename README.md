# resume-data

一个专业的简历数据清洗Web应用程序，专门用于处理和清洗各种格式简历中的各类数据信息。

---

## 📋 项目概述

本项目旨在构建一个功能完善的简历数据清洗系统，支持多种文件格式的简历解析、智能数据提取、自动清洗脱敏以及多格式导出。

### 核心功能

- **多格式支持**：PDF、Word、TXT、HTML、图片（OCR）
- **智能解析**：NLP技术自动识别简历中的各类信息
- **数据清洗**：格式标准化、去重、纠错、脱敏
- **批量处理**：支持批量上传和处理简历
- **多格式导出**：JSON、Excel、CSV等格式

## 📊 数据类型与清洗标准

### 支持处理的简历数据类型

| 数据类别 | 具体字段 |
|---------|---------|
| **个人基本信息** | 姓名、性别、年龄/出生日期、籍贯、民族、政治面貌、婚姻状况、联系电话、电子邮箱、居住地址 |
| **教育背景** | 学校名称、学历、学位、专业、入学时间、毕业时间、教育类型、是否全日制 |
| **工作经历** | 公司名称、部门、职位、工作性质、入职时间、离职时间、工作描述 |
| **项目经验** | 项目名称、项目角色、项目时间、项目描述、技术栈、职责范围 |
| **技能证书** | 技能类型、语言能力、专业证书、培训经历 |
| **奖项荣誉** | 奖项名称、获奖时间、获奖级别、颁奖机构 |
| **自我评价** | 个人简介、职业规划 |
| **期望工作** | 期望职位、期望薪资、工作地点、到岗时间 |

### 数据清洗规则

- **姓名**：去除前后空格，统一中英文格式
- **手机号**：统一格式为 1xx-xxxx-xxxx，校验有效性
- **邮箱**：统一小写，校验格式
- **日期**：统一为 YYYY-MM-DD 格式
- **教育经历**：按时间倒序排列，合并重复学校记录
- **工作经历**：按时间倒序排列，标准化公司名称

## 🛠️ 技术栈

### 后端技术

- **Web框架**：FastAPI (Python)
- **任务队列**：Celery + Redis
- **数据库**：PostgreSQL / SQLite
- **文件处理**：PyMuPDF, python-docx
- **NLP处理**：jieba, 正则表达式
- **OCR**：PaddleOCR / EasyOCR (可选)

### 前端技术

- **框架**：React 18
- **UI库**：Ant Design 5
- **状态管理**：React Context / Redux
- **HTTP客户端**：Axios
- **文件处理**：xlsx, react-dropzone

### 部署

- **容器化**：Docker + Docker Compose
- **Web服务器**：Nginx
- **云服务**：支持阿里云、腾讯云部署

## 🚀 快速开始

### 开发环境

```bash
# 克隆项目
git clone <repository-url>
cd resume-data

# 安装后端依赖
pip install -r requirements.txt

# 启动后端服务
uvicorn main:app --reload

# 安装前端依赖
cd frontend
npm install

# 启动前端服务
npm run dev
```

### Docker部署

```bash
docker-compose up -d
```
